# VibeFi-Ticket-Categorization-Service

Service to categorize tickets based on action and provide expected first hand resolutoin.

ticket_categorizer.py - .py containing the core logic for the service

Instructions to run the file:

1. Installing dependencies - pip install fastapi uvicorn pydantic
2. Running the server: uvicorn ticket_categorizer:app --reload
3. Vsit https link generated with format - <locally_generated_link>/docs
4. Test the endpoint on Swagger UI
